Portfólio: Software

Descrição do Software

O Software é o cérebro do sistema. Ele roda em nossos servidores (nuvem) e é responsável por receber os dados do dispositivo IoT, processar a lógica de irrigação e enviar as notificações para o aplicativo e a plataforma web.
